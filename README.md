# crypto-market-analysis-and-prediction
## Project Objective

The aim of this project is to analyze cryptocurrency prices and market behavior using statistical analysis, visualization, and machine learning models. The project investigates price trends, volatility, feature importance, prediction accuracy, volume-price relationships, anomalies, and seasonal patterns.

## Research Questions

1. What are the short-term and long-term trends in cryptocurrency prices over the observed period?
2. How does price volatility vary over time, and can it be quantitatively modeled?
3. Which available features most significantly influence the cryptocurrency closing price?
4. How accurately can machine learning models predict future cryptocurrency prices?
5. What is the relationship between trading volume and price movements?
6. Can unusual cryptocurrency price spikes and drops be detected as anomalies?
7. Are there recurring weekly or monthly seasonal patterns in cryptocurrency prices?
